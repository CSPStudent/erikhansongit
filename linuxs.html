    Name of Command Description Example
    Is Shows all the sub-folders in a larger folder 'Is/Albums' would show all the folder in the file 'Albums'
    cd Allows user to switch between directories 'cd/blue' would open a directory named 'blue'
    mv Gives the ability to move files between folders 'mv/red/apple /red/banana' would move the file 'red' from folder apple to banana
    man Shows the manual, or information, on the command that is being used 'man Is' would show how the command 'Is' works
    mkdir Makes a new directory 'mkdir orange' would make a directory called orange
    rmdir Removes a directory 'rmdir orange' would remove the diretory named orange
    touch Creates a file 'touch kiwi.doc' would make a document called kiwi
    rm This will remove files from a directory 'rm kiwi.doc' would remove the document kiwi when you are in it's directory
    locate Helps locate where a file is based off of key words 'locate *red*' would look for files with the name 'red' in them
    clear This will clear the screen of whatever there is and take the user back to the starting prompt of the diretory they are in Simply type 'clear' to wie the screen
